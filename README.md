# 🧠 Técnicas Abordadas

---

## 🔹 1. Análise de Componentes Principais (PCA)

### 📁 Pasta:
`Analise de Componentes Principais`

### 📌 Conceitos Trabalhados:

- Matriz de correlação
- Teste de Esfericidade de Bartlett
- Medida KMO (Kaiser-Meyer-Olkin)
- Extração de fatores
- Autovalores (Eigenvalues)
- Variância explicada
- Rotação de fatores
- Interpretação dos componentes principais

### 🎯 Objetivo

Reduzir dimensionalidade mantendo a maior variância possível dos dados.

---

## 🔹 2. Análise de Correspondência

### 📁 Pasta:
`Analise de Correspondencia`

### 📌 Conceitos Trabalhados:

- Tabela de contingência
- Distância Qui-Quadrado
- Inércia
- Autovalores
- Mapa perceptual
- Interpretação das dimensões

### 🎯 Objetivo

Analisar associação entre variáveis categóricas e representar graficamente seus relacionamentos.

---

## 🔹 3. Clustering e Análise de Agrupamentos

### 📁 Pasta:
`Clustering e Analise de Agrupamentos`

### 📌 Conceitos Trabalhados:

- K-Means
- Método do Cotovelo
- Distância Euclidiana
- Centroides
- Segmentação por similaridade
- Interpretação de clusters

### 🎯 Objetivo

Agrupar observações semelhantes em clusters homogêneos.
